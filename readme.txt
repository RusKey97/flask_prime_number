#####
Минипроект на Flask: анализ числа на простоту

ФУНКЦИИ:
    is_prime        - определить, простое число или составное
    prev_prime      - найти предыдущее простое число
    next_prime      - найти следующее простое число
    range_prime     - найти диапазон простых чисел, в котором заключено число
    decompose       - разложить число на простые множители

Поддерживается обработка post запросов (см. post_request.py)
